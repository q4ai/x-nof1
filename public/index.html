Available Placeholders

  • {{STRATEGY_ID}} Strategy identifier
  • {{ENTRY_PROMPT}} Default entry prompt fragment
  • {{EXIT_PROMPT}} Default exit / risk prompt fragment
  • {{VAR_PROMPT}} Default variable notes
  • {{TRADING_INTERVAL_MINUTES}} Trading loop interval (minutes)
  • {{MAX_HOLDING_HOURS}} Maximum holding hours
  • {{MAX_HOLDING_CYCLES}} Maximum holding cycles
  • {{MAX_POSITIONS}} Maximum open positions
  • {{EXTREME_STOP_LOSS_PERCENT}} System extreme stop loss (%)
  • {{ACCOUNT_STOP_LOSS_USDT}} Account stop loss (USDT)
  • {{ACCOUNT_TAKE_PROFIT_USDT}} Account take profit (USDT)
  • {{ACCOUNT_DRAWDOWN_WARNING_PERCENT}} Drawdown warning threshold (%)
  • {{ACCOUNT_DRAWDOWN_NO_NEW_POSITION_PERCENT}} Drawdown no-new-position threshold (%)
  • {{ACCOUNT_DRAWDOWN_FORCE_CLOSE_PERCENT}} Drawdown force-close threshold (%)
  • {{SYMBOL_LIST}} Trading symbols list (delimited)
